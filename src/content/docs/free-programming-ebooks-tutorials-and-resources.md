# รวม free programming ebooks, videos and programming resources ฟรีหนังสือ, วิดีโอ และแหล่งความรู้การเขียนโปรแกรม

- ทีมงาน [dotnetthailand.com](https://www.facebook.com/groups/dotnetthailand) ได้รวบรวม link ของ free programming ebooks ที่มีทั้งรูปแบบ pdf, epub, mobi ให้ download หรืออ่านแบบออนไลน์ โดยทำการแบ่งหมวดหมู่เพื่อง่ายต่อการอ่านและค้นหา
- หากท่านใดมี link ของ free ebooks, videos หรือแหล่งความรู้ที่เป็นประโยชน์ ต้องการแบ่งปันต่อ
สามารถ fork repository นี้ แล้วสร้าง PR มาได้เลย
- ถ้าไม่ถนัดสร้าง PR สามารถ inbox ติดต่อ admin โดยตรงที่ group https://www.facebook.com/groups/dotnetthailand ได้เลยครับ

ขอบคุณครับ
